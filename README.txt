===============
Olive
===============
v0.1.0

A simple, crude, yet working discord bot, designed to poll AlphaVentures API for the stats of a certain Equity, and then plot a graph and post the graph to a discord announce channel.

Dependecies done with poetry.

Do not run on 3.7+ due to an issue with discord.py, alternatively google on how to resolve it by upgrading aiohttp seperately.
